# Bennett Vale Estate 🍷

A luxurious, fully responsive, one-page website built for a premium vineyard and winery. The site features a sophisticated dark-mode aesthetic, dynamic responsive layouts, and smooth micro-interactions.

## Features ✨

- **Modern & Premium Design:** Sleek typography, minimalist layout, and curated high-end imagery.
- **Fully Responsive:** Adapts seamlessly to all devices (desktop, tablet, mobile) with a custom sliding hamburger menu for smaller screens.
- **Smooth Animations:** Utilizes Intersection Observer for elegant fade-up animations as you scroll down the page.
- **One-Page Navigation:** Smooth scrolling directly to sections (Collections, Heritage, Wine Club).

## Technologies Built With 🛠️

- **HTML5**: Semantic structure.
- **Vanilla CSS3**: Advanced CSS properties including Grid, Flexbox, `clamp()` for fluid typography and spacing, and CSS variables for a consistent theme.
- **Vanilla JavaScript**: Lightweight DOM manipulation for the mobile menu toggle and scroll-based fade animations.

## Getting Started 🚀

To view the project locally:

1. Clone this repository.
2. Open the project folder in your favorite code editor (like VS Code).
3. Use an extension like **Live Server** to run `index.html` on a local development server.
